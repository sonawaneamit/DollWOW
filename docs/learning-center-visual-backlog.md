# DollWow Learning Center Visual Backlog

Last updated: 2026-06-28

Use this list for future GPT Image 2 or designer-generated visual assets. Do not block article publishing on these visuals unless the page depends on the image to explain the concept.

## Production Rules

- Prefer vertical mobile-first assets for educational guides: `1024x1536`.
- Use text inside the image when it improves usefulness, but inspect every word before publishing.
- Avoid decorative visuals that do not teach anything.
- Avoid off-category sex toys, dildos, explicit scenes, school themes, underage-coded styling, fake reviews, or claims that imply a specific product result.
- Use actual product references only when provenance is approved.
- For care and option guides, make the visual instructional: steps, comparison grids, warning callouts, or decision trees.
- Generate one sample and inspect it before any batch.
