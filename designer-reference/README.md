# DollWow Designer Reference

Static HTML/CSS snapshots for design review. These files intentionally avoid real product images, Shopify data, scripts, and app logic. Product media areas use abstract placeholder blocks so a designer can focus on layout, visual system, spacing, typography, component rhythm, and interaction direction.

## Files

- `home.html` - homepage direction with hero, comparison card, trust strip, product cards, and feature bands.
- `shop.html` - catalog grid direction with filters and repeated product cards.
- `product.html` - product page and custom build studio direction.
- `styles.css` - standalone CSS tokens and component styling.

## Notes For Design

- The live app uses a premium black, ivory, warm gold, and restrained green palette.
- The product configurator should feel like a polished builder, not a basic Shopify variant form.
- Future option groups are brand/supplier-specific and may need image swatches, price deltas, and conditional conflict states.
- Please do not treat placeholder blocks as final product-media art direction. They are intentionally blank-safe stand-ins.
