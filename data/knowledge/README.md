# DollWow Knowledge Base

This folder is the curated source of truth for AI-assisted features.

Use it for:

- Help Me Choose recommendations.
- Semantic search explanations.
- Product-page spec notes.
- Price-match policy wording.
- Support assistant grounding.
- Factory-photo approval and order concierge copy.

Rules:

- Keep claims factual, short, and reviewable.
- Do not copy competitor wording.
- Do not invent reviews, customer quotes, delivery promises, or supplier approvals.
- If a fact is uncertain, write it as a review note instead of public copy.
- AI features may summarize these notes, but deterministic code still decides discounts, eligibility, policy, and checkout behavior.
